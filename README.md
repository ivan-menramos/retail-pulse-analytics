Objetivo general: Construir un pipeline de analítica end-to-end sobre datos de e-commerce, que cubra desde la ingesta
y limpieza de datos hasta la visualización ejecutiva, simulando un flujo de trabajo real de un equipo de datos.

Objetivos específicos: 
- Diseñar e implementar un esquema relacional en PostgreSQL, con buenas prácticas de modelado.
- Procesar y limpiar datos crudos con python usando pandas, generando un dataset confiable.
- Modelar las transformaciones de negocio con dbt, aplicando capas staging y marts, tests de calidad de datos y generación de lineage graph.
- Construir un dashborad ejecutivo en Power BI con KPIs de ventas, clientes y productos, conectado directamente al modelo de dbt.
